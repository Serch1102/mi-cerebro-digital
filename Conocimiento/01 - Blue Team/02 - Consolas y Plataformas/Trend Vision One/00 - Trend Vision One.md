# Trend Vision One

## DEF
Trend Vision One es una plataforma unificada de operaciones de seguridad que centraliza visibilidad, detección, investigación y respuesta.

## USE
Su principal beneficio es reducir complejidad al concentrar varias funciones de seguridad en una sola plataforma.

## USE
La plataforma aprovecha datos de:
- sensores y productos nativos Trend,
- productos y herramientas de terceros,
- integraciones de seguridad adicionales.

## USE
Recoge telemetría principalmente mediante agentes y sensores, y la combina con otras fuentes para obtener contexto más amplio.

## USE
La información es correlacionada para construir alertas, investigaciones y casos más completos.

## WARN
No se limita solo a productos Trend.
Tampoco sustituye automáticamente todas las demás herramientas del entorno.
